When should I use a class or an ID?

Summary: Best practice is to use classes with CSS and IDs as hooks for Javascript.

Classes vs ID
Class ID
Can be applied to multiple HTML elements Unique - can only be applied to one element
Multiple classes can be applied to the same element Only one ID can be applied to an element
Give an adjective as a name eg. bold-text Give a noun as a name eg. footer or head-nav
Cannot be used as URL navigation Can be used as the URL in an anchor to navigate to a particular section of a page
CSS rules take precedence (have specificity) over elements CSS rules take precedence (have specificity) over classes and elements
